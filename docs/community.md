# Community

OZTP is an open source project. Development happens in the open on GitHub, and contributions are welcome from anyone who shares the mission of making Zero Trust security accessible.

---

## GitHub

All source code, documentation, and issue tracking lives on GitHub.

- **Organization:** [github.com/oztp-org](https://github.com/oztp-org)
- **Control Platform + Device Agent:** [oztp-org/oztp-control-platform](https://github.com/oztp-org/oztp-control-platform)
- **Discussions:** [GitHub Discussions](https://github.com/oztp-org/oztp-control-platform/discussions)

---

## Contributing

Contributions that align with the [core principles](about.md#our-principles) are welcome — particularly:

- Bug fixes and security improvements
- Additional OS support (Linux server agent, Windows Server agent)
- Framework mappings and assessment content
- Documentation improvements
- Translation

**Before opening a pull request**, please review the [MVP Plan](https://github.com/oztp-org/oztp-control-platform/blob/main/docs/oztp-mvp-plan.md) to understand current scope and priorities. A contribution that works well for the current phase is more valuable than one that anticipates a future phase that hasn't started yet.

---

## Code of Conduct

OZTP applies to the following norms:

- Be direct and constructive
- Assume good faith
- Security research must stay within defensive scope — contributions that add offensive capabilities will be declined

---

## Reporting Security Issues

If you discover a security vulnerability in OZTP software, please report it privately rather than opening a public GitHub issue. Use GitHub's private vulnerability reporting feature or contact the maintainers directly via GitHub.

We take security reports seriously and will respond promptly.

---

## Stay Updated

Watch the [oztp-org/oztp-control-platform](https://github.com/oztp-org/oztp-control-platform) repository on GitHub to receive release notifications and follow project progress.
