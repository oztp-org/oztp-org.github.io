# Contributing to the OZTP Website

Thank you for your interest in the Open Zero Trust Project.

## How this project is maintained

The OZTP website is maintained by a small core team. At this stage of the
project, we are **not accepting external pull requests**. All changes are
reviewed and committed by the core team.

This is a deliberate security posture. Software supply chain attacks — where
malicious code is introduced through seemingly legitimate contributions — are
one of the fastest-growing threat vectors in open source. For a project built
around Zero Trust principles, practicing what we preach means applying the same
scrutiny to our own codebase.

## What you can do

- **Open an issue.** Typos, broken links, outdated content, or suggestions are
  welcome via GitHub Issues.
- **Star or share the project.** The best way to support OZTP right now is
  helping others find it.

## Reporting a security vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report privately by emailing the project maintainer directly. We will
acknowledge within 72 hours and work with you on responsible disclosure.

## Questions

General questions are welcome via the chat widget at
[oztp.org](https://oztp.org) or GitHub Discussions.
